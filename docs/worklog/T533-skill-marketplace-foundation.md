# T533 - Bundled skill marketplace foundation

## 1. Task summary

Expand the default skill/template shelf and add a shared catalog foundation for
the future in-app marketplace.

## 2. Repo context discovered

- Default workspace skills are generated by
  `packages/shared/src/workbench/default-workspace-bundle.ts`.
- Stable default skill slugs are declared in
  `packages/shared/src/workbench/bundle-types.ts`.
- New workspaces call the default bundle installer during workspace creation.
- Existing skill loading reads global, workspace, and project skill directories
  from `packages/shared/src/skills/storage.ts`.
- Existing bundled MCP presets already cover Exa, ByteRover, Firecrawl, GitHub,
  Playwright, and Z.AI, so this slice focuses on skills/templates.

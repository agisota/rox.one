# Decision 0050: Continued From Vs Split From

- Status: accepted
- Date: 2026-04-25

## Canonical
```text
continued_from means:
  same authority question
  same canonical frame
  same locus line extended

split_from means:
  materially branched authority problem
  derived locus can resolve separately
```

## Why
- Separates continuity from branching without overloading one lineage edge.

# Decision 0047: Decision Locus Continuation Split

- Status: accepted
- Date: 2026-04-25

## Canonical
```text
continue the same decision_locus when:
  the authority question is unchanged
  and the canonical frame is unchanged

create new_or_split locus when:
  the problem-space materially branches
  and separate resolution paths
  are now possible
```

## Why
- Prevents identity churn for small updates while preserving real branching history.

# Decision 0046: Decision Locus Definition

- Status: accepted
- Date: 2026-04-25

## Canonical
```text
decision_locus:
  is the canonical authority frame
  that groups one or more decision requests
  around the same user-owed decision question

locus owns:
  foreground authority semantics
  lineage
  resolution state
```

## Why
- Gives the authority problem a stable object above individual request surfaces.

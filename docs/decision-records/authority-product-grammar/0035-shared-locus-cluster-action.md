# Decision 0035: Shared Locus Cluster Action

- Status: accepted
- Date: 2026-04-25

## Canonical
```text
shared_locus_cluster action model:
  offer shared actions first
  when one choice can legitimately apply
  across multiple members

member-specific actions remain
drilldown-level actions
```

## Why
- Keeps the cluster surface aligned with shared authority instead of exploding back into member-level duplication.

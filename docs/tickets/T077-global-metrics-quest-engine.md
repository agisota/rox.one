Status: DONE

## Goal

Make metrics and gamification app-wide and evidence-backed instead of cosmetic.

## Scope

- Shared quest graph for ROX Experience progression.
- Runtime-derived quest progress from Experience events.

- UI projections for Progression and Quest Map from the shared truth layer.

- `Прогресс` and `Карта квестов` reflect real app actions.

# Agent Workflow Architect

For workflow tasks:
1. Define input schema.
2. Define output schema.
3. Define state machine.
4. Define required roles/skills/tools.
5. Define validation gates.
6. Write schema/contract tests first.
7. Add fixture-based tests for common workflows.

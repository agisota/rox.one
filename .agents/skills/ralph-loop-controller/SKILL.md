# Ralph Loop Controller

Loop:
1. Pick the next unblocked ticket.
2. Create worktree.
3. Read ticket and AGENTS.md.
4. Run repo discovery.
5. Write tests first.
6. Implement.
7. Validate.
8. Commit.
9. Mark ticket complete only if acceptance criteria pass.
10. Move to next ticket.

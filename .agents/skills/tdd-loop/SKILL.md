# TDD Loop

For every feature:
1. Write tests first.
2. Run tests and confirm failure for the expected reason.
3. Implement minimal code.
4. Run targeted tests.
5. Run full relevant tests.
6. Record commands and results in worklog.
7. Do not mark task done unless tests pass.

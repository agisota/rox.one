# Repo Cartographer

Before writing code:
1. Inspect package scripts, app structure, tests, configs, and relevant modules.
2. Identify existing patterns.
3. Identify likely extension points.
4. Produce a short context map in `docs/worklog/<TASK>.md`.
5. Do not modify source code.
6. Recommend tests before implementation.

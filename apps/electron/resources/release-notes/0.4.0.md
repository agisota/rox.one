# v0.4.0 — LLM-подключения, поддержка Codex, настройки рабочих пространств

Мультиподключение к AI, поддержка Codex/OpenAI, настройки по умолчанию для каждого рабочего пространства и обновление документации.

---

## 🔌 LLM-подключения (несколько провайдеров)

ROX ONE теперь поддерживает несколько **LLM-подключений**:

- Добавление нескольких провайдеров (Anthropic, OpenRouter, Codex/OpenAI и др.)
- Независимая проверка и управление каждым подключением
- Сессии привязываются к подключению после первого сообщения

## 🤖 Поддержка Codex / OpenAI

Подключение OpenAI через **Codex** с использованием OAuth и запуск сессий на базе Codex параллельно с подключениями Anthropic.

## 🧭 Подключение по умолчанию для каждого рабочего пространства

Рабочие пространства теперь могут задавать собственное **LLM-подключение по умолчанию**, перекрывая глобальное значение для новых сессий в этом пространстве.

## 🎨 Темы по умолчанию для каждого рабочего пространства

Каждое рабочее пространство может задать **тему по умолчанию** через `defaults.colorTheme`, что упрощает визуальное различение контекстов.

## 📚 Документация

Новая и обновленная документация:

- LLM-подключения (новая справочная страница)
- Настройка Codex/OpenAI
- Множественные подключения и привязка сессий
- Подключение и тема по умолчанию для каждого рабочего пространства

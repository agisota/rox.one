## Возможности

### Упрощённая OAuth-авторизация
- ROX теперь использует исключительно собственный OAuth-поток для авторизации
- Удалено определение и импорт токенов Claude CLI/Desktop
- Пользователи с устаревшими токенами получат запрос на повторную авторизацию (один раз)
- Более чистый и надёжный процесс авторизации

### Инструменты для тестирования разработчика
- Новый скрипт `test-token-refresh.ts` для тестирования и отладки OAuth-токенов
- Поддержка входа, симуляции истечения срока, определения источника и тестирования миграции
- Добавлен флаг `--token-only` в скрипт fresh-start для тестирования онбординга без потери данных рабочего пространства

## Улучшения

### Логирование OAuth-токенов
- Операции обновления токенов теперь логируют метки времени истечения для упрощения отладки
- Улучшенная видимость жизненного цикла токена и поведения обновления

### Надёжность кэша иконок
- Исправлена обработка null для отсутствующих иконок рабочего пространства
- IPC-вызовы теперь корректно возвращают null вместо выброса исключения для отсутствующих изображений
- Добавлены исчерпывающие тесты для пограничных случаев кэша иконок

## Исправления

- Исправлен сбой автоактивации для источников с `authType: none` (например, Context7)

- Исправлено изменение группировки сессий по датам после перезапуска приложения путём отдельного сохранения `lastMessageAt`
- Исправлены пустые плейсхолдеры документации путём копирования ресурсов документации в Electron dist
- Исправлены сборки OSS `electron:dist:mac` добавлением недостающих скриптов в allow-list
- Исправлен потенциальный сбой при отсутствии файла иконки рабочего пространства
- Исправлена обработка ошибок IPC для отсутствующих изображений рабочего пространства

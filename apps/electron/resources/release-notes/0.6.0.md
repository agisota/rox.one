# v0.6.0 — Встроенный браузер, инструменты документов и мультипанельность

---

## Интегрированный браузер

ROX теперь имеет полноценный встроенный браузер — без расширений, без внешних инструментов. Агент может просматривать веб, заполнять формы, извлекать данные и делать снимки экрана — всё из приложения.

- **Панели браузера в приложении** — Окна браузера открываются как панели рядом с вашим разговором, так что вы можете наблюдать за навигацией агента в реальном времени
- **Одобрение привилегированного выполнения** — Чувствительные действия браузера требуют явного одобрения, оставляя вас под контролем
- **Умные скриншоты** — JPEG по умолчанию для более быстрых и компактных снимков с правильным масштабированием DPI
- **Полный набор CLI-команд** — Навигация, клики, заполнение форм, прокрутка, скриншоты, выполнение JS, управление загрузками и многое другое

### Почему это круто
Это открывает настоящую сквозную автоматизацию. Подключите любой источник данных и позвольте агенту действовать — заполнять формы, отправлять отчёты, навигировать по дашбордам и выполнять многошаговые рабочие процессы полностью без участия. Комбинируйте действия браузера с подключёнными источниками: «извлеки метрики этой недели из Amplitude, заполни форму еженедельного отчёта во внутреннем инструменте и отправь саммари в Slack». Браузер закрывает разрыв между данными и действием.

## Инструменты документов

Встроенные инструменты для работы с PDF, Word, Excel, PowerPoint, изображениями и другими форматами — без пакетов Python или внешних зависимостей.

- **markitdown** — Универсальный конвертер документов в markdown (docx, xlsx, pptx, pdf, html, ipynb)
- **pdf-tool** — Извлечение текста, объединение, разделение и проверка PDF-файлов
- **xlsx-tool** — Чтение, запись и экспорт электронных таблиц Excel
- **docx-tool** — Создание и редактирование документов Word
- **pptx-tool** — Просмотр и работа с файлами PowerPoint
- **img-tool** — Изменение размера, конвертация и извлечение метаданных из изображений
- **doc-diff** — Сравнение двух документов бок о бок
- **ical-tool** — Чтение и парсинг файлов календаря

### Почему это круто
Больше никакого «установи pandas» или «pip install python-docx» — манипуляция документами просто работает. Попросите агента «объединить эти 5 PDF в один», «конвертировать этот Word-документ в markdown», «изменить размер всех изображений в этой папке до 800px по ширине» или «сравнить эти два контракта и подсветить различия». Комбинируйте с источниками для рабочих процессов: «скачай последний отчёт из Google Drive, извлеки таблицу данных и создай задачу в Linear для каждого пункта действий».

## Ветвление и мультипанельный макет

Рабочее пространство теперь поддерживает ветвление разговоров и одновременный просмотр нескольких панелей.

- **Ветвление сессий** — Создайте ответвление разговора в любой точке для исследования разных подходов без потери оригинальной нити
- **Мультипанельный просмотр** — Открывайте браузер, документы и разговоры бок о бок
- **Новый дизайн списка сессий** — Более чистое и организованное управление сессиями

### Почему это круто
Исследуйте две разные стратегии реализации параллельно, держите панель браузера открытой во время программирования или сравнивайте результаты бок о бок. Ветвление означает, что вам больше не нужно беспокоиться о «движении по неверному пути» — просто создайте ветку и попробуйте оба варианта.

## Улучшения

- **Базовый уровень безопасности режима исследования** — Список разрешений по умолчанию теперь включает распространённые команды typecheck (`bun run typecheck`, `tsc --noEmit` и др.) наряду с существующими read-only командами поиска
- **Надёжность инструментов сессий** — Выровнены списки инструментов и их выполнение для всех бэкендов (Claude, Pi, session-mcp-server) для предотвращения расхождений
- **Обработка режима разрешений** — Улучшено отслеживание состояния сессии при переходах между режимами разрешений
- **Область видимости рабочего пространства** — Широковещательные рассылки источников и навыков теперь ограничены активным рабочим пространством
- **Обнаруживаемость автоматизаций** — Добавлены «Узнать больше» и «Добавить автоматизацию» для пустого состояния автоматизаций, а также ссылка в меню справки
- **Надёжность флагов функций** — Безопасное вычисление во время выполнения предотвращает проблемы порядка импорта в debug-сессиях
- **Покрытие typecheck** — `typecheck:all` теперь включает `session-tools-core`

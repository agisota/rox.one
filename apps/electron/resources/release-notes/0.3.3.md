# v0.3.3 — Обновление токенов и исправления автообновления

Автоматическое обновление OAuth-токенов для MCP-источников и значительное улучшение надёжности автообновления.

---

## Автоматическое обновление токенов для MCP OAuth

OAuth-токены для MCP-источников теперь обновляются автоматически при истечении срока.

### Как это работает

**Бесшовная аутентификация**
Когда срок действия вашего OAuth-токена истекает, ROX автоматически обновляет его в фоновом режиме с помощью refresh-токена. Повторная ручная авторизация не требуется — ваши MCP-подключения остаются активными.

---

## Много заголовков аутентификации

API-источники теперь могут использовать несколько заголовков для аутентификации.

### Что нового

**Гибкие заголовки аутентификации**
Настройте источники, требующие нескольких заголовков авторизации (например, API-ключ + ID тенанта). См. обновлённую документацию в `~/.rox/docs/sources.md`.

## Улучшения

### Автообновление
- Исправлено обнаружение обновлений на Windows и macOS
- Корректная обработка пути кэша на Windows (LOCALAPPDATA)
- Обнаружение уже загруженных обновлений при проверке новых версий
- Платформенно-зависимое отображение прогресса с надёжной видимостью кнопок
- Проверка существующих загрузок в обработчике update-available

### OAuth и безопасность
- Поддержка обнаружения метаданных защищённого ресурса по RFC 9728
- Усиление защиты от SSRF при обнаружении OAuth
- Улучшенная обработка ошибок для пограничных случаев OAuth
- Страница успешной авторизации OAuth теперь включает deeplink для более плавного потока

### Производительность
- Параллельная проверка токенов для ускорения запуска
- Ленивая загрузка bundled-документации исправляет инициализацию упакованного приложения

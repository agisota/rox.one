# v0.3.0 — Красивые диаграммы

Рендеринг диаграмм Mermaid в виде красивых SVG. Сверхбыстро, полная поддержка тем, без зависимостей от DOM. Создано для эпохи AI.

---

## Нативный рендеринг Mermaid

ROX теперь рендерит диаграммы нативно с помощью beautiful-mermaid — нового движка рендеринга Mermaid с открытым исходным кодом. Никаких внешних зависимостей, никакого браузерного DOM, чистый TypeScript.

### Почему это важно

Диаграммы необходимы для программирования с помощью AI. Claude постоянно генерирует диаграммы Mermaid — обзоры архитектуры, потоки данных, конечные автоматы, связи сущностей. Раньше это были просто блоки кода. Теперь это **красивые, профессиональные визуализации**.

### Что нового

**Встроенный рендеринг**
Блоки кода Mermaid в ответах автоматически рендерятся как нативные SVG-диаграммы. Никакой настройки не требуется.

**Полноэкранный предпросмотр**
Нажмите на любую диаграмму, чтобы открыть её в просмотрщике в стиле Figma:
- **Зум колёсиком мыши** к курсору (адаптируется к мыши или тачпаду)
- **Щипок на тачпаде** для плавного пропорционального масштабирования
- **Перетаскивание кликом** для панорамирования
- **Двойной клик** для сброса до 100%
- **Горячие клавиши**: `Cmd/Ctrl +/-` зум, `Cmd/Ctrl 0` сброс
- **Выпадающий список пресетов**: 25%, 50%, 75%, 100%, 150%, 200%, 400% и «Подогнать по размеру»

**Типы диаграмм**
- Блок-схемы (`graph LR/TD`)
- Диаграммы состояний (`stateDiagram-v2`)
- Диаграммы последовательностей (`sequenceDiagram`)
- Диаграммы классов (`classDiagram`)
- ER-диаграммы (`erDiagram`)

**Оформление**
Элегантная двухцветная основа — цвета фона и переднего плана определяют все визуальные элементы через CSS `color-mix()`. Диаграммы автоматически соответствуют вашей теме ROX.

**Производительность**
100+ диаграмм рендерятся менее чем за 500 мс. Чистый TypeScript, нулевые зависимости от DOM.

## Система предпросмотра в приложении

Помимо диаграмм, этот релиз представляет унифицированную систему предпросмотра для расширенного контента.

### Предпросмотр PDF

Просмотр PDF в приложении с использованием Mozilla pdf.js:
- Рендеринг на основе canvas с выделением текста
- Элементы навигации по страницам
- Декодирование в фоновом потоке

### Предпросмотр изображений

Нативный просмотр распространённых форматов:
- PNG, JPG, GIF, WebP, SVG, BMP, ICO, AVIF
- HEIC/HEIF и TIFF открываются системным просмотрщиком

### Умное определение файлов

Клик по пути к файлу теперь автоматически открывает нужный предпросмотр:

| Тип | Расширения | Поведение |
|------|------------|----------|
| Изображение | png, jpg, gif, webp, svg... | Предпросмотр в приложении |
| PDF | pdf | Предпросмотр в приложении |
| Код | ts, js, py, rs, go, java... | Предпросмотр с подсветкой синтаксиса |
| Markdown | md, mdx | Форматированный предпросмотр |
| JSON | json, jsonc | Сворачиваемое дерево |

## Улучшения

### Рендеринг Markdown
- **Блоки кода JSON** отображаются как интерактивные сворачиваемые деревья
- **Блоки кода Mermaid** отображаются как нативные SVG-диаграммы

- Новый резолвер сопоставляет распространённые CLI-инструменты (git, npm, bun, cargo, docker, python, node...) с соответствующими иконками.

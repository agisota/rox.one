# Что нового в v0.9.0

- Claude Agent SDK: приложение перешло на native-binary distribution и корректно упаковывает бинарь для macOS, Linux и Windows.
- Messaging: добавлена интеграция Lark / Feishu с личными чатами, группами, карточками действий, файлами и изображениями.
- Telegram: supergroup можно привязать к workspace, а automation-задачи маршрутизировать по topic.
- Сессии: появился режим группировки списка по непрочитанным и прочитанным.
- Messaging Settings: карточка Telegram получила отдельные секции Direct и Supergroup.
- Сессии: загрузка сообщений стала надёжнее, вместо бесконечного spinner показывается Retry.
- Сборка: Bun linker закреплён в `hoisted`, а Docker/WebUI зависимости объявлены явно.
- Исправлены: API-ключи loopback custom endpoints, vision overrides, Mermaid-валидация, stale reconnect и iOS OAuth popups.

# v0.8.8 — Исправления локальных моделей, кнопка повтора и межсессионный обмен сообщениями

## Возможности
- **Межсессионный обмен сообщениями** — Новый инструмент `send_agent_message` позволяет сессиям отправлять сообщения другим активным сессиям, обеспечивая координацию между сессиями и рабочие процессы agent-to-agent.

## Улучшения
- **Переводы на китайский (zh-Hans)** — Обновлены и исправлены переводы на упрощённый китайский по всему UI.
- **Документация локальных моделей** — Уточнено требование пути `/v1` для эндпоинтов локальных моделей.

## Исправления
- **Упрощена настройка локальных моделей** — Настройка локальных моделей больше не запрашивает API-ключ, когда он не нужен; placeholder-ключ передаётся автоматически для эндпоинтов, не требующих аутентификации. Также исправлено отображение модели в селекторе.
- **Кнопка повтора** — Кнопка повтора теперь корректно повторно отправляет последнее сообщение пользователя и правильно управляет фокусом ввода чата.
- **Дублирование ConfigWatcher на headless-сервере** — Исправлено дублирование рекурсивных ConfigWatcher на headless-серверах Bun/Linux, которое могло вызывать периодические зависания после активности сессии.
- **Обработчик уровня размышления** — IPC-обработчик `SET_DEFAULT_THINKING_LEVEL` теперь корректно возвращает успешный ответ.

## Критические изменения
- Нет.

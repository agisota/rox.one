# v0.8.6 — Поблочная передача сессий и поддержка изображений для пользовательских эндпоинтов

## Возможности
- **Поблочная передача сессий** — Большие сессии теперь можно передавать между рабочими пространствами через поблочный WebSocket RPC с кодированием base64, контрольными суммами SHA-256 и повторными попытками для каждого блока. Позволяет избежать лимитов размера WebSocket-сообщений, которые ранее приводили к тихим сбоям передачи. Прогресс передачи отображается как фиолетовая анимация LED-рамки на кнопке «Отправить» с нормализованным отслеживанием прогресса для многопоточных пакетных операций.
- **Ввод изображений для пользовательских эндпоинтов** — Модели пользовательских эндпоинтов (например, **Gemma 4** через Ollama) теперь могут получать вложения изображений. Ранее изображения тихо отбрасывались, поскольку пользовательские модели были жёстко настроены на текстовый ввод. **Важно:** пользовательские эндпоинты по умолчанию остаются текстовыми — необходимо установить `supportsImages: true` в конфигурации подключения для включения ввода изображений.

## Исправления
- **Усиление восстановления передачи сессий** — Улучшено восстановление после ошибок при передаче между рабочими пространствами с независимым декодированием блоков base64, увеличенным размером блоков (2 МБ) и надёжной обработкой переподключений.
- **Передача использовала неверный ID рабочего пространства** — Экспорт сессий теперь корректно использует ID исходного рабочего пространства вместо целевого, исправляя некорректную маршрутизацию при передачах.
- **Запросы учётных данных во встроенном чате** — Запросы учётных данных и разрешений теперь корректно работают внутри EditPopover встроенного чата.
- **Потеря сообщений после спящего режима/пробуждения** — Дополнительное исправление для исчезновения сообщений при переподключении по устаревшему WebSocket после спящего режима/пробуждения ПК.
- **Макет кнопки отправки** — Восстановлены правильные отступы между кнопками «Отмена» и «Отправить» для соответствия паттернам диалогового footer.

## Критические изменения

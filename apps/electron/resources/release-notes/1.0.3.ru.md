# Что нового в v1.0.3

## Основное

- Stable-релиз теперь публикует агрегированный `manifest.json`, `release-notes.json` и установочные скрипты `install-app.sh` / `install-app.ps1` вместе с macOS, Windows и Linux артефактами.
- Публичный feed `https://app.rox.one/electron/stable/manifest.json` указывает на v1.0.3 и содержит контрольные суммы для `ROX-ONE-arm64.zip`, Linux AppImage и `ROX-ONE-x64.exe`.
- Packaged startup smoke для macOS проходит: приложение больше не застревает на первом loading screen из-за renderer chunk regression.
- Rox Design payload впервые включен в macOS package как external ASAR payload под `app.asar.unpacked/resources/rox-design`.

## Известные ограничения

- macOS build v1.0.3 остается ad-hoc signed и не notarized. Gatekeeper может блокировать quarantined install до появления Developer ID signed/notarized release.
- Rox Design payload присутствует в v1.0.3, но packaged runtime resolver в этом релизе ищет старый путь `Contents/Resources/app/resources/rox-design`. Source hotfix подготовлен для следующего patch release; до перепаковки v1.0.3 не следует считать Rox Design полностью исправленным для всех пользователей.

## Следующий patch

- Следующий release должен включить packaged resolver fix для `app.asar.unpacked/resources/rox-design` и подписанный/notarized macOS artifact перед тем, как объявлять Mac launch полностью закрытым.

# v0.4.5 — Исправление Slack-токенов и сборка Copilot

Срочное исправление, решающее проблему неработоспособности источников Slack из-за некорректной обработки системой обновления токенов нерerefreshable OAuth-токенов, а также улучшения сборки CLI Copilot.

---

## Исправления

- Исправлена поломка нерerefreshable OAuth-токенов (Slack) функцией `ensureFreshToken()` — токены без `refreshToken` (например, токены Slack-ботов) теперь возвращаются как есть вместо перехода к обновлению, которое завершается сбоем и помечает источник как `needs_auth` (e358e414)
- Исправлена неотчистка кулдауна обновления при повторной авторизации — после повторной авторизации источника 5-минутный кулдаун теперь немедленно очищается, чтобы источник был доступен без ожидания (e358e414)
- Исправлен Copilot CLI на Windows — добавлен `copilot.exe` в `extraResources` для избежания блокировки файлов EBUSY при упаковке electron-builder (d4cda5f2)
- Добавлен `copilot` в валидатор схемы типов LLM-провайдеров (d4cda5f2)

## Сборка и CI

- Добавлена проверка бандла Copilot CLI в рабочие процессы CI для всех платформ (macOS arm64/x64, Linux, Windows) (d4cda5f2)
- Нативный бинарный файл Copilot CLI теперь корректно поставляется для каждой платформы в скриптах сборки с проверкой размера (d4cda5f2)

# Что нового в v0.8.12

- GPT-5.5 стал моделью по умолчанию для OpenAI и OpenAI Codex подключений.
- DeepSeek добавлен как поддерживаемый Pi-backed provider с корректными defaults.
- `source_test` теперь может включить источник и перезапустить turn, чтобы tools стали доступны сразу.
- `spawn_session` принимает `thinkingLevel` для дочерних сессий.
- `pi-agent-server` получил настоящий typecheck через `tsc --noEmit`.
- WebUI больше не запускает desktop app при добавлении новой label.
- Attachments надёжно сохраняются между сессиями и не теряются при переключении.
- Исправлены: custom URL schemes, `/compact` timeout, регистрация Pi tools, `call_llm` model routing, mini completions и malformed drafts.

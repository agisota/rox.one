Мгновенный поиск по всем сессиям. Мини-агенты для быстрых целенаправленных правок.

## Поиск по сессиям

Находите что угодно в истории разговоров с помощью полнотекстового поиска на базе ripgrep.

### Как это работает

**Быстрый доступ**
Нажмите `Cmd/Ctrl + F` или кликните иконку поиска в боковой панели.

**Умные результаты**
- Поиск ведётся только по сообщениям пользователя и ассистента (фильтрует системный шум)
- Показывает фрагменты совпадений прямо в списке сессий
- Подсвечивает совпадения в реальном времени при вводе

**Навигация**
- Используйте кнопки-шевроны для перехода между совпадениями
- Автопрокрутка к совпавшему тексту в разговоре
- Счётчик совпадений показывает общее количество по всем сессиям
- Автоматически разворачивает разбитые на страницы ходы для отображения ранних совпадений

**Производительность**
- Поиск на базе ripgrep для мгновенных результатов даже с тысячами сессий
- Подсвечиваются только видимые совпадения для плавной прокрутки

## Мини-агенты

Легковесные агенты для быстрых изменений конфигурации без накладных расходов полного Claude Code.

### Что нового

**Целенаправленное редактирование**
Мини-агенты используют компактный системный промпт, оптимизированный для конкретных задач, таких как редактирование статусов и меток. Никаких лишних инструментов или контекста.

**Быстрее и дешевле**
Настраиваются на использование более быстрых моделей (например, Haiku) для простых задач, не требующих полной мощности рассуждений.

**UI встроенного выполнения**
Новый компактный индикатор прогресса показывает, что делает агент, прямо внутри поповеров:
- Последние 3 действия с описаниями
- Пути к файлам и обрабатываемые паттерны
- Сообщения об успехе/ошибках в формате markdown

**Перетаскиваемые поповеры**
Поповеры редактирования теперь имеют рукоятку для перетаскивания — перемещайте их в любое место экрана.

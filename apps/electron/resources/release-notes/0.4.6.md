# v0.4.6 — HTML/PDF-превью, адаптеры событий и исправления OSS-ошибок

Крупный релиз с добавлением встроенного рендеринга HTML и PDF-превью, новой унифицированной архитектуры адаптеров событий для поддержки нескольких бэкендов, массовых операций с сессиями и исправлений нескольких проблем, сообщённых сообществом.

---

## Возможности

- **Встроенные HTML и PDF-превью** — новые блоки кода `html-preview` и `pdf-preview` отображают HTML-контент и PDF-документы прямо в чате с поддержкой вкладок для нескольких элементов и полноэкранным просмотром (0d22069b)
- **Контекстное меню массовых операций с сессиями** — правый клик при множественном выборе теперь показывает массовые операции (статус, метки, флаг, архивирование, удаление) вместо действий для одной сессии (61759915)
- **Динамическое обнаружение моделей Codex** — получение доступных моделей с app-server Codex с периодическим обновлением каждые 30 минут и fallback на жёстко заданный реестр при сбое. Поддержка нескольких подключений одного типа провайдера с уникальной генерацией slug (1bcb1083)
- **Шаблонизатор Mustache** — новая шаблонная система для динамического рендеринга контента с синтаксисом `{{variable}}`, партиалами, условиями и циклами (a779bf83)
- **Шаблоны источников** — источники теперь могут поставлять HTML-шаблоны для фирменного, согласованного рендеринга своих данных через новый инструмент `render_template`. Шаблоны используют синтаксис Mustache с самоописывающимися заголовками метаданных (992a1a50)
- **Флаг режима Fast** — перенесён в общий пакет для кросс-пакетной доступности (e8295580)

## Исправления

- **Пустая история чата при запуске через вход в macOS** — добавлен затвор инициализации на базе Promise в SessionManager; обработчик `GET_SESSIONS` теперь ожидает завершения инициализации для предотвращения возврата пустых результатов во время стартовой гонки (9ed91fbf)
- **Окно не перетаскивается по всем заголовкам панелей** — меню приложения, панель списка сессий и заголовки панелей теперь имеют согласованные области перетаскивания, чтобы окно можно было перемещать из любой области заголовка (e1ed6619)
- **Навыки не могут автоматически активировать необходимые источники** — добавлено поле `requiredSources` в frontmatter SKILL.md; навыки теперь объявляют зависимости от источников, которые автоматически включаются до первого хода агента (9ed91fbf)
- **Ссылки на файлы в чате не кликабельны** — csv, pdf, png и другие типы файлов не обрабатывались обработчиком кликов по ссылкам. Список расширений теперь извлекается из `file-classification.ts` для автоматической синхронизации (a3fa35e4)
- **source_test сообщает об истечении для действительных OAuth-токенов** — теперь выполняется попытка обновления токена перед сообщением об истечении, что соответствует поведению фактического конвейера подключений (7a9bc9d9)
- **Прерванные ответы вызывают путаницу LLM** — частичный контент от остановленных/перенаправленных ответов теперь получает инъекцию контекстной заметки, чтобы модель не интерпретировала последующие сообщения некорректно (8fd1eb7d)
- **Слишком крупные иконки в селекторах рабочих процессов и выпадающих списках** — исправлен размер иконок в компонентах StyledDropdown и селекторе рабочих процессов (2c917b88, aef87854)
- **Путь Git Bash на Windows не сохраняется** — обнаруженный `CLAUDE_CODE_GIT_BASH_PATH` теперь сохраняется в конфигурацию и восстанавливается при запуске (9ed91fbf)
- **Состояние гонки переопределения окружения для отдельных сессий** — глобальная изменяемая `optionsEnv` заменена на `envOverrides` для каждой сессии для исправления гонки `ANTHROPIC_BASE_URL` при параллельных сессиях (1bcb1083)

## Улучшения

- **Унифицированная архитектура адаптеров событий** — общая логика потоковой передачи/событий извлечена в `BaseEventAdapter` с бэкенд-специфичными адаптерами для Claude, Codex и Copilot. Включает новую `EventQueue` для упорядоченной доставки событий (несколько коммитов)
- **Переименование TodoState в SessionStatus** — согласованные имена в 54+ файлах с обратной совместимостью для существующих конфигураций хуков и миграции файлов сессий (e5987124)
- **Улучшение интервалов между блоками** — лучшее визуальное расстояние между блоками контента в чате (a0402076)
- **Улучшения таймаута инструмента Transform** — более надёжная обработка таймаутов для скриптов преобразования данных (c03011b6)

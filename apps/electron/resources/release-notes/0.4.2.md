# v0.4.2 — Поддержка Copilot, переработка Mermaid, таблицы данных

Крупные новые возможности: GitHub Copilot как альтернативный провайдер, полностью перестроенный движок диаграмм Mermaid, встроенные таблицы данных и множество исправлений стабильности.

---

## 🤖 Поддержка GitHub Copilot

- Добавлен GitHub Copilot как альтернативный LLM-провайдер с полной OAuth-аутентификацией
- Новые экраны подключения для настройки и переключения между провайдерами
- Индивидуальные иконки для Claude, Copilot, OpenAI, Ollama, OpenRouter и Vercel
- Предупреждения о привязке сессии при переключении между провайдерами

## 📊 Встроенные таблицы данных и электронные таблицы

- Новый блок `datatable` для сортируемых и фильтруемых отображений данных
- Новый блок `spreadsheet` для таблиц в стиле Excel с поддержкой экспорта
- Инструмент `transform_data` для обработки больших наборов данных через скрипты
- Таблицы с файловым хранилищем через поле `src` для наборов данных с 20+ строками

## 📐 Переработка движка диаграмм Mermaid

- Замена движка компоновки Dagre на ELK для улучшенного расположения графов
- Улучшена маршрутизация ребер, связывание и направления подхода
- Поддержка многострочных меток во всех типах диаграмм
- Новая система фигур: круги, ромбы, шестиугольники, стадионы и формы состояний
- Улучшения рендеринга ASCII с поддержкой цветов ANSI
- Улучшены переопределения направлений подграфов и обработка несвязных графов

## 🔧 Улучшения подключений и провайдеров

- Уточнены экраны настройки LLM-подключений
- Обновление до Claude Opus 4.6 в качестве модели по умолчанию
- Улучшения обновления OAuth-токенов
- Поддержка множественных заголовков аутентификации для API-источников

## 🔌 Исправления интеграции Codex

- Исправлено разрешение пути к встроенному бинарнику Codex в упакованных приложениях
- Перемещение codex.exe в extraResources на Windows для предотвращения ошибок EBUSY
- Внедрение содержимого навыков в сессии Codex (больше не зависит от нативного обнаружения)
- Исправления тихих сбоев и генерации заголовков для OAuth-сессий
- Проверка сборки теперь проверяет наличие бинарника Codex на этапе комплектации SDK

## 🐛 Исправления

- Исправлена ошибка ENOENT при запуске OAuth-браузера с использованием shell.openExternal
- Исправлено отсутствие обновления темы рабочего пространства при переключении
- Исправлены пути к бейджу дока и иконкам окон в упакованном приложении
- Исправлено: Sentry отправляет только ошибки консоли, а не предупреждения
- Исправлен скрипт установки PowerShell для использования YAML-манифестов
- Исправлен импорт модуля Node.js fs в рендерере внутри браузерного бандла
- Исправлено состояние гонки метаданных при сохранении сессий
- Консолидирована обработка больших ответов

## 📦 Обновления зависимостей

# v0.4.3 — Система хуков, инструмент LLM и исправления стабильности

Добавлена комплексная система хуков для автоматизации рабочих процессов по событиям приложения и агента, новый инструмент `call_llm` для вторичных AI-вызовов и важные исправления стабильности.

---

## 🪝 Система хуков

- Новая событийно-ориентированная архитектура хуков для выполнения команд и внедрения промптов по событиям приложения и агента
- Шина событий с поддержкой жизненного цикла сессий, изменений разрешений, обновления меток и тактов планировщика
- Командные хуки выполняют shell-команды с безопасностью на основе разрешений и песочницей
- Промпт-хуки внедряют контекст в разговоры агента при срабатывании событий
- Планирование на основе cron через новый SchedulerService
- Ограничитель частоты событий и fallback SIGKILL для неконтролируемых процессов хуков
- Метаданные сессии доступны как переменные окружения для скриптов хуков
- Автоактивация источника при ссылке в промпт-хуках
- Логгер событий для отладки выполнения хуков

## 🧠 Инструмент LLM (`call_llm`)

- Новый инструмент в области сессии для вторичных вызовов Claude для специализированных подзадач
- Поддержка рабочих процессов суммаризации, классификации, извлечения и анализа
- Загрузка файлов и изображений через вложения с поддержкой диапазона строк
- Предопределенные форматы вывода и поддержка произвольных JSON Schema
- Режим расширенного мышления для сложных задач рассуждения
- Поддержка параллельного выполнения нескольких одновременных вызовов

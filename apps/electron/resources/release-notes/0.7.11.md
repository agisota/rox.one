# v0.7.11 — Переключатель контекста 1M и исправления настройки Bedrock

## Возможности
- **Переключатель контекста 1M** — Настройка для каждого рабочего пространства для включения/отключения контекстного окна 1M для Opus 4.6. Отключите для экономии лимитов использования. (Настройки → Рабочее пространство → Дополнительно)

## Улучшения
- **Контекстное окно пользовательской конечной точки** — Модели пользовательских конечных точек теперь могут указывать `contextWindow` в конфигурации модели вместо жёсткого ограничения 128K.

## Исправления
- **Форма настройки Bedrock** — Исправлен сбой настройки подключения Bedrock с ошибкой «Пожалуйста, введите корректный API-ключ» при использовании потока Pi API Key. Выпадающие списки уровней моделей Pi перехватывали отправку до того, как она достигала обработчика Bedrock.
- **Контекст 1M Sonnet** — Удалён суффикс `[1m]` у Sonnet 4.6, который ещё не поддерживается, предотвращая потенциальные проблемы с ограничением частоты запросов.
- **Усечение имени модели** — Исправлено отображение имён моделей пользовательских конечных точек (например, `gpt-5.4`) как просто «Gpt» в выпадающих списках. Номера версий теперь сохраняются.

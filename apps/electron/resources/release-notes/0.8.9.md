# v0.8.9 — Opus 4.7 по умолчанию

## Возможности
- **Claude Opus 4.7 как новая модель по умолчанию** — Anthropic выпустила Claude Opus 4.7 16 апреля 2026 года, и теперь это модель Opus по умолчанию во всём приложении. Существующие пользовательские конфиги автоматически мигрируются с 4.6 на 4.7 при запуске. Opus 4.6 остаётся доступным, а записи Bedrock для него сохранены для обратной совместимости.

## Улучшения
- **Claude Agent SDK обновлён до 0.2.111** — Добавляет нативную поддержку Opus 4.7 и новые возможности SDK: `mcp_set_servers` с `permission_policy` для каждого инструмента, публичный API `startup()` / `WarmQuery` и новое событие `system/memory_recall`. Включает обновления безопасности上游 (`@anthropic-ai/sdk` обновлён до ^0.81.0).

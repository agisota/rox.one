## Возможности

### Управление просмотром Diff
- Новая панель управления для просмотра diff с переключением split/unified режима
- Переключатель подсветки фона для удобного чтения diff
- Встроенная статистика добавлений (+) и удалений (-) с первого взгляда
- Настройки сохраняются через `preferences.json` (заменяет localStorage)

### Улучшения для Windows
- Обнаружение и настройка Git Bash для пользователей Windows
- Автоматическое обнаружение Git Bash в стандартных путях установки
- Возможность ручного выбора для нестандартных установок Git Bash
- Понятные инструкции по настройке для Windows

### Расширенное меню приложения
- Полное выпадающее меню в стиле macOS с подменю
- Стандартные операции редактирования: Отменить, Повторить, Вырезать, Копировать, Вставить, Выделить всё
- Управление окнами: Свернуть, Развернуть, Новое окно
- Управление видом: Увеличить, Уменьшить, Сбросить масштаб
- Подменю отладки доступно в dev-сборках

## Улучшения

### Кроссплатформенная надёжность
- Новые правила ESLint, требующие кроссплатформенных паттернов кода
- Обработка путей переработана для использования платформенно-зависимых разделителей
- Обработка временных директорий использует пути, соответствующие ОС
- Разрешение путей в монорепозитории корректно работает на всех платформах

### Модернизация системы сборки
- Скрипты сборки перенесены с shell-команд на TypeScript
- Отдельные скрипты для main, preload, renderer и ресурсов
- Улучшен dev-сервер с поддержкой HMR
- Улучшенная поддерживаемость и совместимость с Windows

### Сохранение сессий
- Асинхронная персистентность на основе очереди для надёжной записи
- Счётчик генерации обработки предотвращает перезапись устаревших сообщений
- Более надёжное управление состоянием сессий
- Улучшенная обработка параллельных операций

### Визуальная polish
- Централизованная стилизация фона для всех оверлеев предпросмотра
- Визуальные улучшения оверлеев и состояний «в процессе»
- Улучшенная архитектура компонентов оверлеев
- Большая согласованность между preview diff, кода и терминала

# Что нового в v0.8.7

- Добавлены венгерский, немецкий и польский переводы UI.
- Польская локаль получила корректные формы множественного числа.
- API-источники теперь поддерживают `renewEndpoint` для обновления короткоживущих токенов.
- API proxy умеет отправлять raw body через `_rawBody` и `_contentType`.
- Исправлены defaults и region binding для Bedrock models.
- Model dropdowns теперь прокручиваются при длинных списках.
- Файлы рабочей директории workspace открываются корректно.
- Server lock освобождается при выходе и устойчивее восстанавливается после crash.
